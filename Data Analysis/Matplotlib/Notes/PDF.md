# 📊 Probability Density Function (PDF): From Intuition to Insight

> **Core Concept:** A PDF describes how probability is distributed across possible values of a **continuous random variable**.  
> **Hinglish Intuition:** *"PDF basically batata hai: Continuous data me values kaha zyada likely hain aur kaha kam likely hain."*

---
